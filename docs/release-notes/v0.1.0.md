# 灯语助手 v0.1.0

首个公开版本，提供可在安卓手机上运行的闪光灯控制与灯语发送功能。

## 新增功能

- 延时关灯：支持设置小时和分钟，确认开灯后立即亮灯并显示倒计时
- 闪灯模式：支持自定义闪灯间隔，开始与停止控制
- 灯语模式：
  - 国际摩斯码输入，自动转换为灯光节奏
  - 自定义长闪、短闪、符号间隔、词间隔
  - 循环播放次数设置
  - 手动停止发送
- 前台服务：保证闪灯和灯语任务在后台继续执行

## 已知说明

- `app-release-unsigned.apk` 为未签名包，不能直接作为正式安装包分发
- `app-release.aab` 可作为后续正式签名或商店发布的基础产物
- 某些国产系统可能对后台闪光灯和精确定时有额外限制

## 产物

- Release AAB：`app/build/outputs/bundle/release/app-release.aab`
- Release APK：`app/build/outputs/apk/release/app-release-unsigned.apk`
